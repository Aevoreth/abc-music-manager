# Settings — Default filters

**Settings → Default filters**

Configure default values for [Library filters](../library.md#filters).

---

## When defaults apply

- When you first open the app
- When you click **Reset Filters** on the Library page

**Clear Filters** uses the most permissive values (show everything), not these defaults.

---

## Options

Mirrors the Library filter bar:

- In set (Either / Yes / No)
- Rating range
- Duration min/max
- Last played (time or date range)
- Parts count min/max
- Default status and transcriber selections

Adjust defaults if you usually work with a subset of your library (e.g. only unrated songs).
