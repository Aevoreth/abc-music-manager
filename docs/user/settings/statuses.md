# Settings — Statuses

**Settings → Statuses**

Custom status labels for library organization (e.g. "Needs practice", "Retired", "Event ready").

---

## Managing statuses

- **Add status** — name and color
- **Edit** / **Delete** existing statuses

Each song has one status at a time. Change status from the [Library](../library.md) Status column or [Song detail](../song-detail-and-layouts.md).

Statuses appear in the Library status filter chips.
