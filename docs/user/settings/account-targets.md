# Settings — Account targets

**Settings → Account targets**

Define where **File → Write PluginData…** writes `SongbookData.plugindata` files.

---

## Scan Account Targets

After setting the [LOTRO directory](folder-rules.md#lotro-directory), click **Scan Account Targets**.

The app discovers account PluginData paths under your LOTRO installation.

---

## Enable / disable

Use the **Enabled** column to choose which accounts receive updates on each write.

---

## Manual targets

**Add account target** for custom paths (advanced / non-standard installs).

---

See [PluginData / Songbook](../plugindata.md) for the full write workflow.
