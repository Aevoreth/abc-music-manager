# Settings — ABC Playback

**Settings → ABC Playback**

Defaults for the [playback toolbar](../playback.md).

---

## Soundfont path

Path to an SF2 soundfont file. **Empty** uses automatic lookup (MaestroCommon) or prompts to download.

- **Browse…** — pick a file
- **Use default** — clear the path

---

## Default volume

0–100, applied when playback starts.

---

## Default tempo

0.5× to 2×. The toolbar tempo control can override per session.

---

## Stereo mode and width

| Mode | Behavior |
|------|----------|
| **Band layout** | Pan from player positions on the active band layout |
| **Maestro: user-pan** | Use user pan directives from ABC |
| **Maestro: Default** | Maestro default panning |

**Stereo width** (0–100): 0 = full left/right separation, 100 = centered.
