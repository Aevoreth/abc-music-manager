# ABC Music Manager — User Guide

ABC Music Manager helps *Lord of the Rings Online* player musicians organize ABC music libraries, build setlists for events, model band layouts, manage multi-part playback for yourself or live over the internet, and write Songbook data for your game accounts.

> **Tip:** Access this documentatioon at any time using **Help → User Guide** (or the **User Guide** button on an empty library) Links below open other topics in this guide.

## Main areas of the app

| Page | What it is for |
|------|----------------|
| **Library** | Browse, filter, and play your indexed ABC songs. See [Library](library.md). |
| **Setlists** | Build event setlists, timing, and exports. See [Setlists](setlists.md). |
| **Bands** | Define players, instruments, and band layouts. See [Bands](bands.md). |
| **Set Play** | Run a live set as bandleader. See [Set Play](set-play.md). |
| **Band Assistant** | Follow along as a band member. See [Band Assistant](band-assistant.md). |
| **Settings** | Paths, filters, playback, relays, and more. See [Settings → Folder rules](settings/folder-rules.md). |

## Need help right now?

**Library is empty?** Complete [First-time setup](getting-started.md#getting-started-checklist) — set your LOTRO folder, scan the library, and check that filters are not hiding everything ([Library → filters](library.md#filters)).

**No sound when playing?** See [ABC Playback settings](settings/abc-playback.md) and [Troubleshooting → Playback](troubleshooting.md#playback-no-sound).

**Preparing for a gig?** Start with [Setlists](setlists.md), then [Set Play](set-play.md) if you want live coordination.
