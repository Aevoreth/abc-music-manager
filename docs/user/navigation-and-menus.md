# Navigation and menus

---

## Left navigation panel

The main window has a vertical nav list on the left:

| Page | Purpose |
|------|---------|
| **Library** | Song table, filters, playback |
| **Setlists** | Setlist Folders, sets, set export, ABCP import/export |
| **Bands** | Bands, players, layout grids |
| **Set Play** | Bandleader live set control |
| **Band Assistant** | Connect to a leader's relay session |
| **Settings** | Preferences, configuration, and paths |

Use **View** in the menu bar to jump directly to any page.

---

## File menu

| Item | Action |
|------|--------|
| **Scan Library** | Index ABC files from configured folders |
| **Analyze duplicate folders…** | Find folder structures that mirror each other |
| **Settings** | Opens the Settings page |
| **Write PluginData…** | Write Songbook data to enabled account targets |
| **Exit** | Quit the application |

---

## View menu

Lists every main page (Library, Setlists, Bands, Set Play, Band Assistant, Settings) for quick navigation.

---

## Help menu

| Item | Action |
|------|--------|
| **User Guide** | Opens this documentation |
| **About** | Version, license summary, third-party components |

---

## Playback toolbar

A persistent toolbar at the top of the main window controls playback: transport, volume, tempo, parts/playlist, scrub bar, stereo, and band layout selection.

See [Playback toolbar](playback.md).

---

## Unsaved changes

Some pages (Setlists, Bands, Settings) track unsaved edits. Switching away or closing the app may prompt you to save or discard changes.
