# Band Assistant

**Band Assistant** lets band members follow a bandleader's Set Play session over a relay connection.

---

## In-app vs standalone {#standalone-mode}

- **In app:** open **Band Assistant** from the left nav

---

## Connecting {#connect}

1. Leader must enable **Broadcast** on [Set Play](set-play.md#broadcast) and share a room code
2. Choose the same **Relay** as the leader ([Set Playback settings](settings/set-playback-relays.md))
3. Enter the **Room** code
4. Click **Connect**

Use **Reconnect** after connection drops. **Disconnect** leaves the session.

---

## What syncs {#what-syncs}

Assistants receive the leader's Set Play state:

- Current / next / skip / played flags
- Loaded setlist identity
- Band layout grid and part-change highlighting

---

## What does not sync

Selecting player names to highlight on the grid is **local to each client** (leader or assistant) and is not broadcast. Useful to remind user of which player(s) they are controlling for visual reference.
